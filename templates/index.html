Welcome

This application demonstrates a pipeline for processing DICOM files:

  • Extracts metadata from 50 mock DICOM files on startup.
  • Maps metadata to ROO and SNOMED CT ontologies.
  • Generates RDF triples and stores them in an in-memory graph.
  • Provides a SPARQL endpoint for querying the data.
  • Includes a Health DCAT-AP styled metadata catalog.
  • Visualizes the resulting Knowledge Graph.
